# Claude Code チュートリアル

Claude Code を実務で活用できるレベルになるための日本語チュートリアルです。
「コマンドの説明」ではなく「どう使えば開発が速くなるか」を軸に解説します。

## 対象読者

- Claude（Web版）は使ったことがあるが、CLI / エディタ拡張は未経験の方
- SIer・Web 系企業への就職を控えた学生・院生

## サンプルプロジェクト

全章を通じて **TypeScript + Express の TODO API** を使います。
→ [`sample-project/`](./sample-project/)
